# ExploCubes!

A casual match-3 puzzle game built in **Unity**.  
Combine colorful cubes, trigger chain reactions, and use rockets to clear obstacles!

![Gameplay Preview](https://github.com/redempkaan/ExploCubes/blob/main/ExploCubes.gif)


## Mechanics
- Tap to match **3 or more cubes** of the same color  
- Create **rockets** from larger matches  
- Chain reactions between horizontal & vertical rockets  
- Different **obstacles** like boxes, stones, and vases  
- Level-based progression system with **move limits**

## Tech Stack
- **Engine:** Unity 6000.0.32f1  
- **Language:** C#  
- **Animation:** DOTween
